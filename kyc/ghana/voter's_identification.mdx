## You can verify your customers identity using their passport identification number

The Voters Identification check using ZeeH's endpoint helps you to get in touch with their information on a verified national level.


### Request Endpoint

Generates user data using Voters ID

**POST** `{{url}}/ghana_kyc/lookup_voters_id`

#### Request Fields

```json
{
    "type": "MAIN",  // OLD
    "vin": 9001332866
}
```
