## You can verify your customers identity using their National Identification Basic

The National ID Basic check using Adhere's endpoint helps you to get in touch with their information on a verified national Identification.


Request Endpoint

**POST**
 `{{url}}/kenya_kyc/lookup_national_id_basic`

#### Response Fields
```json
{
    "firstName": "john",
    "lastName": "doe",
    "nationalID": 12345678
}
```
