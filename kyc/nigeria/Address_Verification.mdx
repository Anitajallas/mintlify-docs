## Verify your customers' Address

Adhere's Address verification endpoint provides access to accurate and verified information about your customers'address.

#### Request Endpoint

`POST` `{{BASE URL}}/api/onboarding/nigeria_kyc/address_verification/`

#### Sample Payload
``` json
{    
    "city": "oshodi",
    "lga": "lagos mainland",
    "landmark": "lagos",
    "street": "270 Murtala Muhammed Way, Alagomeji. Yaba",

    "last_name": "Doe",
    "birth_date": "17/01/1988",
    "phone_number": "08000000000",
    "nin": "10000000001",
    "state": "Lagos"

}
```
