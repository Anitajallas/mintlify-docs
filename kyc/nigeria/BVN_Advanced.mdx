## Ensures Customers Are Who They Claim to Be

The BVN Endpoint provides a set of fields to verify a user's Bank Verification Number (BVN). This endpoint is useful for KYC (Know Your Customer) and user verification purposes. It can also be used for auto-completing account data and reducing fraud. The endpoint returns personal information such as name, phone number, date of birth, national identification number (NIN), local government area of origin, state of origin, image, and more. The BVN helps identify the user from a specific financial institution.


### Advanced BVN Lookup

This endpoint generates a more detailed BVN response object.

#### Request Endpoint

Generates user data from BVN.

`POST` `{{url}}/onboarding/nigeria_kyc/bvn_advanced`

```json
{
    "bank_verification_number": "22000000001"
}
```
