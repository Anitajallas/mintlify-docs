## Ensures Customer's Identity Verification

The BVN Endpoint offers a set of fields to verify the user's Bank Verification Number (BVN). This endpoint is essential for Know Your Customer (KYC) and user verification purposes. It can also be used to auto-complete account data and reduce fraud. The endpoint returns personal information such as name, phone number, date of birth, national identification number (NIN), local government area of origin, state of origin, image, and more. The BVN helps identify the user from a specific financial institution.

### Advanced BVN Lookup With Date of Birth

This endpoint generates a detailed BVN response object based on BVN and Date of Birth.

** POST **
Endpoint: `{{url}}/nigeria_kyc/lookup_bvn_advance_igree`

#### Sample Request

```json
{
    "bvn": "22220000000", 
    "dateOfBirth": "2002-04-01"
}
```
