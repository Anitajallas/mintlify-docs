## Ensures Customers Are Who They Claim to Be

The BVN Endpoint provides a set of fields to verify a user's Bank Verification Number (BVN). This endpoint supports KYC (Know Your Customer) and user verification purposes. It can also be used for auto-completing account data and reducing fraud. It provides personal information such as name, phone number, date of birth, national identification number (NIN), local government area of origin, state of origin, image, and more. The BVN helps identify the user from a specific financial institution.

### Simple BVN Lookup With Date of Birth

This endpoint generates a concise BVN response object.

#### Request Endpoint

Generates user data from BVN and Date of Birth.

**POST**

`{{url}}/nigeria_kyc/lookup_bvn_basic_igree`



#### Sample Request

```json
{
    "bvn": "22200000000",
    "dateOfBirth": "2002-04-01"
}
```

200 OK Response
```json
{
  "success": true,
  "statusCode": 200,
  "message": "Bank Verification Number Details Retrieved",
  "response_code": "00",
  "data": {
    "firstName": "ABRAHAM",
    "middleName": "OLADIRAN",
    "lastName": "FATOKI",
    "dateOfBirth": "01-April-2002",
    "phoneNumber": "08130000500"
  }
}
```
