### Simple CAC Lookup

This endpoint generates a concise description of the details of the application.

#### Request Endpoint
**Generate simple customer data object**

**POST** `{{url}}/nigeria_kyc/lookup_cac`

#### Payload
``` json
{    
    "rc_number": "1261103",
    "company_name": "ADAFRUIT LIMITED"
}
```
