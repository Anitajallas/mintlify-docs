### Advanced CAC Lookup

These endpoints fetch more advanced information about an account. They retrieve a list of financial accounts associated with a specific app's public key.

#### Request Endpoint
**Generate advanced customer data object**

**POST** `{{url}}/nigeria_kyc/lookup_cac_advanced`


#### Payload
```json
{    
    "rc_number": "1261103",
    "company_name": "ADAFRUIT LIMITED"
}
```
