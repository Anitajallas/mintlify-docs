## Fetch Address Using NIN

Adhere's Address endpoint provides access to fetch an address using your customers' information.

#### Request Endpoint

`POST` `{{BASE URL}}/api/onboarding/nigeria_kyc/identity_address/`

#### Sample Payload
``` json
{    
    "first_name": "John",
    "last_name": "Doe",
    "time_range": 2, # minimum  time range (in months) that the address must've been verified eg 6 for 6 months, 2 for 2 months
    "phone_number": "08000000000",
    "nin": "10000000001",
    "state": "Lagos"
}
```
