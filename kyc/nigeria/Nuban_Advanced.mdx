You can verify your customers' identity using their NUBAN Number.

The NUBAN check using Adhere's endpoint helps you access their information on a verified national level.


**POST:** `{{url}}/nigeria_kyc/lookup_nuban_advance`


### Status Codes

- **200:** OK. Data is successfully returned.
- **400:** Bad Request. The NUBAN parameter is invalid. Request is not completed successfully.
- **404:** Not Found. NUBAN parameter is not found. Request is not completed successfully.
