### Response Fields

| Field             | Description                                           | Type    |
|-------------------|-------------------------------------------------------|---------|
| `status`          | This is the returned status of the request, either success or failed. | `String` |
| `code`            | This is the operational code.                         | `Number` |
| `data`            | This is an object containing every transaction data.   | `Object` |
| `data.uuid`       | A unique id value for the license data.               | `String` |
| `data.licenseNo`  | The licensee Number value.                            | `String` |
| `data.firstName`  | First Name of the verified holder.                    | `String` |
| `data.lastName`   | Last Name of the verified holder.                     | `String` |
| `data.middleName` | Middle name of the verified holder.                   | `String` |
| `data.gender`     | Gender of the verified holder.                        | `String` |
| `data.issuedDate` | Date the license was issued.                         | `String` |
| `data.expiryDate` | Expiry date of the card.                             | `String` |
| `data.stateOfIssue` | State in which the license card was issued.        | `String` |
| `data.birthDate`  | Birth Date of the verified holder.                    | `String` |
| `data.photo`      | Photo of the verified holder in base64 format.        | `base64` |
