### User Verification using the National Identification Number (NIN)

This endpoint allows you to fetch and verify customer details using the National Identification Number (NIN). The NIN provides robust information about an individual.

#### Request Endpoint
**Generate user data from their NIN**

**POST** `{{url}}/nigeria_kyc/lookup_nin`

#### Payload
```json
{
    "nin": "70123456789"
}
```
