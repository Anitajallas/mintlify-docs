You can verify your customers' identity using their Phone Number.

This helps to verify customers using their phone numbers.

### Basic Phone Lookup

#### Request Endpoint
**Generate simple customer phone data object**

**POST** `{{url}}/nigeria_kyc/lookup_phone_number_basic`


#### Response Fields
- **Sample request:**
  ```json
  {
      "phone_number": "09011111111"
  }
  ```
Response Codes:

200: OK - Represents a successfully completed request
404: Not Found - Record not found. Indicates the phone number passed might be invalid
400: Bad Request - The phone number parameter is invalid. Request is not completed successfully
