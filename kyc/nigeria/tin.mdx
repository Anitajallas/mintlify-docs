# Verify Customer Identity Using Their Tax Identification Number (TIN)

You can verify your customers' identity using their Tax Identification Number (TIN) through the API.

## Payload

```json
{
    "tin": "18609323-0001"
}
```

## Response

```bash
{
    "success": true,
    "statusCode": 200,
    "message": "Tax Identification Number Details retrieved",
    "response_code": "00",
    "data": {
        "search": "07012345678",
        "taxpayer_name": "JOHN DOE NIG LTD",
        "cac_reg_number": "RC123456",
        "firstin": "12345678-0001",
        "jittin": "N/A",
        "tax_office": "MSTO ALIMOSHO",
        "phone_number": "07012345678",
        "email": ""
    }

```

## Response Fields

| **Field**            | **Description**                                                        | **Type**  |
|----------------------|------------------------------------------------------------------------|-----------|
| **status**           | This is the returned status of the request, either success or failed.  | `String`  |
| **code**             | This is the operational code.                                          | `Number`  |
| **data**             | This is an object that contains all the transaction data.              | `Object`  |
| **data.tin**         | TIN of the verified taxpayer.                                          | `String`  |
| **data.taxpayer_name** | Full name of the verified taxpayer.                                  | `String`  |
| **data.cac_reg_number** | Registration number of the verified taxpayer.                      | `String`  |
| **data.firstin**     | TIN of the verified taxpayer.                                          | `String`  |
| **data.jittin**      | Joint Tax Board number of the verified taxpayer.                       | `String`  |
| **data.tax_office**  | Tax Office of the verified taxpayer.                                   | `String`  |
| **data.email**       | Email of the verified taxpayer.                                        | `String`  |
| **data.phone_number** | Phone Number of the verified taxpayer.                                | `String`  |
