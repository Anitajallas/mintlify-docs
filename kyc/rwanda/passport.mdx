## Verify Your Customer's Identity Using Their Passport
The API provides an endpoint to verify customer identity using passport information. 
This endpoint allows you to retrieve details associated with a verified passport, ensuring a streamlined and reliable identity verification process.

## Request Endpoint
The endpoint generates user data based on the provided passport number.


**Method:** `POST`  
**Endpoint:** `/rwanda_kyc/lookup_passport`  
**Media Type:** `application/json`


Payload Fields

```json
{
    "passportNumber": "PD000000"
}
```
